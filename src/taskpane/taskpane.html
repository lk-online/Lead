Please note:

  • Clicking a button and entering information doesn't send an email automatically; you still need to click on "send"
  • Unless you complete all required fields, the OK button of the modal will be disabled
  • If the font size of the produced email body is wrong, simply select it and set it manually
  • With the exception of the font size, editing the email body directly will most probably break the format; don't do it
  • If you need to send a highly customized message, do it manually
  • If you are unsure about something you did, it's better to discard the message and start anew
